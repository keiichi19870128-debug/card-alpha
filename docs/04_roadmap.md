## 優先順位と判断基準

| 優先度 | 基準 |
|--------|------|
| **P0（最優先）** | MVP機能。なければサービスの体裁がならない |
| **P1（重要）** | 収益化に直結（有料会員） |
| **P2（将来）** | 差別化・競合優位（AI分析、ランキング） |
| **P3（拡張）** | 市場拡大（他TCG対応） |

### 判断フロー

```
コンバージョン（会員登録・課金）に繋がるか？
  ├── Yes → P0 or P1（即座に実装）
  └── No → 差別化要素か？
            ├── Yes → P2（MVP後に実装）
            └── No → P3（市場成長後に実装）
```
