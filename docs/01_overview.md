| 項目 | 内容 |
|------|------|
| サービス名 | Card Alpha |

---

## 2. サービスコンセプト

Card Alpha は、TCG（トレーディングカードゲーム）の市場価格と投資・コレクション価値を分析し、
ユーザーが賢明な判断を下せるように支援するプラットフォームです。

### 差別化ポイント

- **単なる価格検索サイトではない**：独自の考察・分析コンテンツを提供
- **予算別アプローチ**：1万円〜15万円の予算に応じた具体的な戦略を提示
- **将来性を重視**：30周年（2026年）に向けた仕込み情報など、先を読む情報を発信
- **拡張性**：ポケモンカードの仕組みをそのままワンピースカードなど他TCGに流用可能

---

## 3. ターゲットユーザー

| セグメント | 特徴 | ニーズ |
|-----------|------|--------|
| 投資初心者 | カード投資に興味があるが何から始めればいいか分からない | 予算別の具体的な戦略、入門コンテンツ |
| コレクター | 価値の高いカードを集めたい | 注目カード情報、将来性のあるカードの発見 |
| 高騰情報求む | 儲けたい・資産価値を上げたい | 早めの高騰予測情報 |
| 30周年仕込み組 | ポケモン30周年（2026年）に向けて今から準備 | 長期的に価値が上がるカードのリスト |

## 4. 技術スタック

### 推奨構成（Replit対応）

| 層 | 技術 | 選定理由 |
|----|------|---------|
| フロントエンド | Next.js 14 + TypeScript + Tailwind CSS | Replitで動作、SSR対応、モバイル最適 |
| バックエンド | Next.js API Routes（Route Handlers） | フルスタックで1プロジェクトで完結 |
| データベース | PostgreSQL（Replit Database または Supabase） | 無料枠あり、管理画面不要で使いやすい |
| ORM | Prisma | 型安全、マイグレーション簡単 |
| 認証 | NextAuth.js | メール認証・ソーシャルログイン対応 |
| 決済 | Stripe（将来的） | 有料会員の月額課金に対応 |
| デプロイ | Vercel（推奨）または Replit デプロイ | 無料でNext.jsホスティング |

### 補足：なぜこのスタックか

- **Replitで動く**：環境構築が不要、ブラウザだけで開発可能
- **モバイル対応**：Tailwind CSSのレスポンシブ + Next.jsのSSRで高速表示
- **拡張性ユニーク**：カード・ゲーム・カテゴリを抽象化することで、ポケモン→ワンピース対応が容易
- **管理画面不要**：Prisma + Next.js APIで直接データ管理、あるいは簡易CMSとして管理ペーンを作る

## 6. 主要機能マップ

### MVP（Phase 1）

```
Card Alpha
├── ホームページ
│   ├── サービス説明
│   ├── 最新記事一覧（最新5件）
│   ├── 注目カード一覧（最新5件）
│   └── カテゴリナビゲーション（ポケモンカード）
├── 予算別投資戦略

│   └── 15万円コース
├── 注目カード一覧
│   ├── カード詳細（名前・価格・理由・評価S/A/B）
│   └── 絞り込み・ソート

│   ├── カテゴリ：ポケモンカード / 投資戦略 / 高騰予想
│   └── 記事検索
└── 管理画面（簡易）
    ├── 記事作成・編集
    ├── 注目カード登録・編集
    └── 予算別戦略登録・編集
```

### Phase 2（会員・有料機能）

```
├── 会員登録 / ログイン
├── 有料会員プラン
│   ├── 限定記事閲覧
│   ├── 予算別戦略詳細版
│   └── 注目カード完全リスト

```

```

```

## 7. デザイン方針

| 要素 | 方針 |
|------|------|
| トーン | 信頼感・プロフェッショナル、でも親しみやすい |
| カラー | ダークテーマベース＋アクセントカラー（ゴールド/グリーン） |
| タイポグラフィ | 読みやすいゴシック系、数字は等幅フォントで価格表示 |
| モバイル | フルレスポンシブ、タップしやすいボタンサイズ |
| 価格表示 | カンマ区切り、トレンド矢印（↗↘→）で変動を可視化 |

## 8. セキュリティ・運用方針

| 項目 | 対応 |
|------|------|
| 認証 | bcrypt + JWT + セッション管理 |
| 決済 | Stripe Checkout（PCI DSS対応をStripeに委託） |
| 管理者画面 | ロールベースアクセス制御（RBAC） |
| バックアップ | データベースの自動バックアップ（週次） |
| SEO | Next.jsのSSRでメタタグ最適化、構造化データ対応 |
