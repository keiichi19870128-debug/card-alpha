# Card Alpha 画面構成・ワイヤーフレーム設計

## 1. 画面一覧

| No | 画面名 | 種別 | 説明 |
|----|--------|------|------|
| 1 | ホームページ | 公開 | サービス説明・最新情報表示 |
| 2 | 予算別投資戦略一覧 | 公開 | 予算選択画面 |
| 3 | 予算別投資戦略詳細 | 公開 | 選択した予算の戦略・おすすめカード表示 |
| 4 | 注目カード一覧 | 公開 | 登録された注目カードの一覧 |
| 5 | 注目カード詳細 | 公開 | 単一カードの詳細情報 |
| 6 | 記事一覧 | 公開 | カテゴリ別・新着記事一覧 |
| 7 | 記事詳細 | 公開 | 個別記事の本文表示 |
| 8 | 管理画面ダッシュボード | 管理 | 管理トップ・指標サマリー |
| 9 | 記事管理 | 管理 | 記事の作成・編集・削除・一覧 |
| 10 | 注目カード管理 | 管理 | カードの作成・編集・削除・一覧 |
| 11 | 予算別戦略管理 | 管理 | 戦略コンテンツの編集 |

## 2. 共通レイアウト

```
+--------------------------------------------------+
|  [Logo] Card Alpha        [ホーム] [戦略] [注目] [記事]  |
+--------------------------------------------------+
|                                                  |
|              メインコンテンツエリア                |
|                                                  |
+--------------------------------------------------+
|  フッター                                          |
|  ・サイトマップ                                    |
|  ・利用規約・プライバシーポリシー                    |
|  ・SNSリンク                                       |
+--------------------------------------------------+
```

### 共通UIコンポーネント

| コンポーネント | 配置 | 説明 |
|--------------|------|------|
| ヘッダーナビ | 固定トップ | ロゴ + メインメニュー + ハンバーガー（モバイル） |
| フッター | ページ下部 | リンク群 + コピーライト |
| カードコンポーネント | 各所 | 画像 + タイトル + 価格 + 評価バッジ |
| セクションタイトル | 各所 | 左線 + 太字タイトル + 英語サブタイトル |

## 4. モバイル対応

全画面共通で、横幅 768px 以下では以下のレイアウト変更を行います。

| 要素 | PC（≥768px） | モバイル（<768px） |
|------|-------------|------------------|
| ナビゲーション | 横並びリンク | ハンバーガーメニュー |
| カードグリッド | 4列 | 2列 |
| サイドバー | 固定左側 | 非表示 or 上部展開 |
| フォントサイズ | ベース16px | ベース14px |
| ボタンサイズ | height 40px | height 48px（タップしやすく） |
